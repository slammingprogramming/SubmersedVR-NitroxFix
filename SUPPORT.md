# Support

Use this guide to choose the best support path for SubmersedVR-NitroxFix.

## Start Here

- New install: [Installation Guide](docs/INSTALLATION.md)
- Hardware or runtime status: [Compatibility](docs/COMPATIBILITY.md)
- Startup, VR, controller, or server problem: [Troubleshooting](docs/TROUBLESHOOTING.md)
- Current limitations: [Known Issues](docs/KNOWN_ISSUES.md)
- Quick setup questions: [FAQ](docs/FAQ.md)

## Open An Issue

Use the matching issue template:

- [Bug report](.github/ISSUE_TEMPLATE/bug_report.md) for reproducible problems.
- [Compatibility report](.github/ISSUE_TEMPLATE/compatibility_report.md) for tested hardware, runtime, storefront, or controller setups.
- [Documentation issue](.github/ISSUE_TEMPLATE/documentation.md) for README, docs, template, or metadata improvements.
- [Question](.github/ISSUE_TEMPLATE/question.md) for setup or usage questions.

## Include Logs

Press `F11` in-game to generate diagnostics, then attach:

```text
BepInEx/LogOutput.log
```

Include versions for Subnautica, Nitrox, SubmersedVR, SubmersedVR-NitroxFix, BepInEx, SteamVR, your headset, and your controller type.
