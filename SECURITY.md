# Security Policy

## Supported Versions

SubmersedVR-NitroxFix is currently maintained through the latest release on the repository's [Releases](../../releases) page.

## Reporting A Vulnerability

If you believe you found a security issue, please avoid posting exploit details in a public issue.

Instead, contact the maintainers privately if a private channel is available on the repository profile. If no private channel is available, open a minimal [bug report](.github/ISSUE_TEMPLATE/bug_report.md) that describes the affected area without sharing sensitive reproduction details publicly.

Please include:

- Affected version or commit.
- High-level impact.
- Whether the issue requires a malicious server, malicious client, crafted save, or local file access.
- Any safe reproduction notes you can share privately.
